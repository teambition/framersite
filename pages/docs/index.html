layerA = new Layer
layer.x = 500

Class: Layer
Type: property: number

--- ###### layer.screenFrame The absolute position for this layer on the screen. 
layerA = new Layer 
layerA.screenFrame = {x:100, y:300, width:100, height:100}

Class: Layer
Type: property: object

--- ###### layer.addSubLayer(layer) Add a subLayer to this layer.
Options Value Description
layer (required) Layer object The layer that will become a subLayer 
layerA = new Layer 
layerB = new Layer
layerA.addSubLayer(layerB)

Class: Layer
Type: function

--- ###### layer.draggable.calculateVelocity() The current x and y velocity for the dragged layer. The velocity is calculated over the average 0.1 second by default and expressed in pixels per second. 
layerA = new Layer
layerA.draggable.enabled = true

# {x: 5, y: 2}
print layerA.calculateVelocity()

Class: Layer Draggable
Type: function
Returns: object

Property Value Description
x Number x speed in pixels per second
y Number y speed in pixels per second
