{% endblock sidebar %} {% block body %} {% filter pygmentify %} {% filter markdown %} ### Import Designs Framer lets you import entire designs from Photoshop or Sketch with a single click. It fits right into your current design workflow. Continuously update your designs while bringing them to life. --- ##### Get Started Start by clicking the **"Import"** icon in the toolbar. You’ll be presented with the Import sheet. It tells you whether or not you have any file opened that’s ready to be imported. Note that you can only import files that you’ve previously saved.
### Set-up Let’s have a look at how to set-up your files before importing. First, make sure that your layers are grouped and have unique names. This is to ensure that you can select them later. Ungrouped layers are flattened. They either become part of the background or that of their parent group.
##### Selecting layers After you’ve succesfully imported a file, the following line of code gets inserted on top of your code: 
# This imports all the layers of "design" into designLayers
designLayers = Framer.Importer.load "imported/design"
This loads and stores all of your importer layers. We can assign it any variable name, and use that to select layers. Let’s rename it to “import”. Now, let's select a layer group titled “layerA” in Photoshop or Sketch: 
# This imports all the layers of "design" into designLayers
import = Framer.Importer.load "imported/design"

# Set the opacity of layerA 
import.layerA.opacity = 0.5
##### Selecting multiple layers You can also select multiple layers at once. This can be really useful if you’d like to animate or re-position multiple layers, for example. This can be done with a for-loop. This allows you to either select all of your imported layers, or just the layers within a specific group.
