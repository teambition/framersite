{% endblock sidebar %} {% block body %} {% filter pygmentify %} {% filter markdown %} ### Import Designs With Framer, you can bring your designs to life using tools you’re already comfortable with. Import files from Photoshop or Sketch with a single click. Go back-and-forth and start exploring interactions while further refining your visuals. After importing, all of your layers are selectable and have their position, hierarchy and masking already set. You don’t have to start from scratch when looking to quickly validate an idea. --- ##### Get Started Start by clicking the **"Import"** icon in the toolbar. You’ll be presented with the Import sheet. It tells you whether or not you have any file opened that’s ready to be imported. You can only import files that have already been saved once before.
### Set-up There are only a few things you need to keep in mind when looking to import your Photoshop or Sketch files. First, make sure that your layers are grouped and have unique names. This allows you to select them later. Next, if you're using spaces in your layer names, they'll be converted to underscores. Finally, all of your un-grouped layers get flattened. If they're part of a parent layer group, they become part its background.
##### Selecting layers After you’ve successfully imported a file, the following line of code gets inserted at the top of your code: 
# This imports all the layers of "design" into designLayers
designLayers = Framer.Importer.load "imported/design"
This loads and contains all of of the importer layers. We can assign it any name, and use that to select layers. Let’s simply rename it to “import”. Now, to select a layer group that you’ve named “layerA”, you can write: 
# This imports all the layers of "design" into designLayers
import = Framer.Importer.load "imported/design"

# Set the opacity of layerA 
import.layerA.opacity = 0.5
Layer groups placed within other groups are called subLayers. In the example below, "groupA" has two subLayers. They can also be selected directly. There's no need for you to take the layer hierarchy into account.
##### Selecting multiple layers You can also select multiple layers. Perhaps you’d like to animate a bunch of layers, or hide one of your layer groups. This can be done with a for-loop. You can choose to select all of your imported layers, or just the layers of a specific group.
