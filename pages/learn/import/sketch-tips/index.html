{% endblock sidebar %} {% block body %} {% filter pygmentify %} {% filter markdown %} ### Sketch Tips ##### Position on Canvas The x and y position of your layers are turned into x and y properties. In the example below, the layer group Screen is correctly positioned at 0,0 and will appear at the top left in Framer. This also means that layers with *negative* coordinates are positioned outside the canvas. This doesn’t apply when you use Artboards (more below).
##### Hidden Layers Layers that you’ve hidden are also imported, but with visibility set to "false" by default. The opacity property is left unchanged. This feature is currently only available in Sketch Beta.
##### Artboards Only the first artboard will be visible by default. All other artboards are hidden, and positioned on top of eachother. The x and y properties of imported artboards are always set to 0.
