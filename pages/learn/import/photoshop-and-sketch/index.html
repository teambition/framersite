{% endblock sidebar %} {% block body %} {% filter pygmentify %} {% filter markdown %} ### Photoshop & Sketch Every layer group in Photoshop and Sketch will be imported as a layer within Framer. Hidden layers will also be imported, but will also be hidden in Framer, with opacity set to zero.
##### imported layers After import, the following code will be inserted above your code: 
# This imports all the layers for "Prototype" into prototypeLayers
prototypeLayers = Framer.Importer.load "imported/Prototype"
This allows you to target all of the imported layers within Framer.
All of your imported layers are placed within an /imported folder, which are then included in your prototype. Note that the **default variable name** (prototypeLayers in the above example) can be changed to anything. --- ### Principles Framer is fully compatible with Sketch 3.1, Sketch Beta, Photoshop CS6, Photoshop CC and Photoshop CC 2014.
##### Photoshop & Sketch - Only saved & active documents can be imported - Every layer group will become a layer in Framer - Layer group names should be unique - Masked layer groups become layers with clip enabled ##### Sketch only - Only the first artboard will be visible in Framer by default, to show another one you need set the visible property to true and opacity larger than 0. - Set the coordinates of your artboard or first visible object to x:0 and y:0 - Items with negative coordinates will not be visible in Framer - You can only use one mask per layer group
