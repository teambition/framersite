{% endblock sidebar %} {% block body %} {% filter pygmentify %} {% filter markdown %} ### States States are sets of properties. States can be named, and later interacted with. A layer can have multiple states. You can add, remove and override states, as well as animate between them. Every layer has a "default" state with its current properties, which is included when switching states. 
# Add states
layerA.states.add 
    second: {scale:0.75} 
    third:  {rotation:90, scale:1} 
    
# Set the animation options
layerA.states.animationOptions = 
    curve: "spring(600,30,0)"

# Toggle states on click
layerA.on Events.Click, -> 
    layerA.states.next()
Just like layer animations, states also take a set of curves. You can define the animationOptions of all states of a layer at once. 
# Set the animation options
layerA.states.animationOptions = 
    curve: "spring(600,30,0)"
##### Switching states Besides toggling between all states, you can also switch to a specific state, optionally without an animation. 
# Switch to third state
layerA.states.switch("third")

# Switch to third state without animation
layerA.states.switchInstant("third")
##### Editing States You can remove and override previously defined states. To override states, simply add a new state with the same name. 
# Remove second state
layerA.states.remove("second")

# Override third states
layerA.states.add
	third: {blur:10}
Summary
  • States are sets of properties
  • Every layer has a default state
  • You can animate between states
  • States can be added, removed and overriden
