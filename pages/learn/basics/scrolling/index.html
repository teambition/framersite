{% endblock pagetitle %} {% block body %} {% filter pygmentify %} {% filter markdown %} #### Scrolling You can scroll a layer contained within a superLayer, as long as its height or width exceeds that of its parent. 
# Enable scrolling
layerA = new Layer height: 100
layerB = new Layer height: 200, superLayer: layerA

layerA.scroll = true
Alternatively, you can enable scrolling for a specific direction only: 
layerA.scrollHorizontal = true
layerA.scrollVertical = true
There is one scrolling event:
