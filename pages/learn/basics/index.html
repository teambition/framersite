{% endblock pagetitle %} {% block body %} {% filter pygmentify %} {% filter markdown %} ### Layers A layer is a rectangle that draws elements. Layers can display images, videos or text, and contain a set of basic properties. 
# Create a layer & define its properties
layerA = new Layer 
	image: "icon.png", 
	width:128, height:128,
	x:64, y:64, opacity:1,
	scale:1, rotation:90
##### Hierarchy Parent layers are called superLayers, and child layers are called subLayers. Layers inherit properties from their superLayers. You can define a layers superLayer as a property, too. 
# When creating a new layer
layerB = new Layer
	x:20, y:20, superLayer:layerA

# After layers have been created
layerB.superLayer = layerC
layerC.subLayer = layerB
##### Dynamic Properties The values of layer properties can also be dynamic. 
layerA = new Layer 
	x: layerB.x
	y: layerB.y + 40
