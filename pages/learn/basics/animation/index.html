{% endblock pagetitle %} {% block body %} {% filter pygmentify %} {% filter markdown %} #### Animation Most properties of a layer can be animated: x, y, width, height, scale, opacity, etc. You can animate multiple properties at the same time. Animations optionally take a curve that describes the type of animation. You can use pre-defined curves like 'linear' or 'ease-in', custom ones, or spring animations. You can create an animation explicitly with myAnimation = layer.animate({..}) or with the shortcut on any layer layer.animate({..}). 
# Creates an animation using a spring
springUp = layer.animate
	properties: {y: layer.y + 100}
	curve: "spring(100,10,0)"
