{% endblock sidebar %} {% block body %} {% filter pygmentify %} {% filter markdown %} ### Animation Most properties of a layer can be animated: x, y, width, height, scale, opacity, etc. You can animate multiple properties at the same time. 
layerA.animate
	properties:
		opacity: 0.5
##### Timing Except when using spring curves, you can also define the duration of an animation and how often it repeats. 
layerA.animate
	properties:
		opacity: 0.5
	curve: "ease"
    time: 1
##### Curves Animations optionally take a curve that describes the type of animation. You can use pre-defined curves like 'linear' or 'ease-in', custom ones, or spring animations. An overview of different curves: 
# Linear curve
curve:"linear"

# Easing curve
curve:"ease-in-out"

# Easing curve
curve:"spring(200,15,0)"

Summary

  • Most properties of layers can be animated
  • Animations can be timed and repeated
  • Framer contains various animation curves, including spring physics
