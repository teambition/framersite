{% endblock sidebar %} {% block body %} {% filter pygmentify %} {% filter markdown %} ### CoffeeScript Basics CoffeeScript is a more legible and minimal way to write JavaScript. CoffeeScript is an attempt to expose the good parts of JavaScript in a simple way. --- ##### Overview CoffeeScript uses *significant whitespace*. That means that tabs are used to define the structure of the language, instead of **{** and **}** like in Javascript. So if you forget to indent, your script won't work. 
# This will work
if 1 + 1 == 2
	console.log "hello"

# This won't work
if 1 + 1 == 2
console.log "hello"
##### Variables A variable is a way to store something under a name. It's pretty simple: 
myVariable = 1
From this point on the variable myVariable contains 1. You can change this at any point just by assigning a new value. --- #### Functions Functions are bits of executable code. It's pretty much the same thing as a function in mathematics. They take input (arguments) and generate output when they're exectuted (called). This is how you define a function. The *return* keyword passes back a value to the caller. By default, CoffeeScript returns the latest used variable. 
myFunction = (input) -> 
	output = input * 2
	return output
To call this function with 42 as its argument: 
myFunction(42) # Output: 84
You can assign the output of a function to a new variable: 
myCalculatedNumber = myFunction(42)
myCalculatedNumber # Output: 84
A function can be defined with as many arguments as you like and they can be any type: 
myFunction = (argumentA, argumentB, argumentC) ->
	return argumentA + argumentB + argumentC
