{% endblock sidebar %} {% block body %} {% filter pygmentify %} {% filter markdown %} ### Loops Loops are used to iterate over collections, to do something for each item. Collections can be an Array or Object. They use the "for" keyword. 
myArray = [1, 2, 3, 4, 5]

for number in myArray
	number + 1

# Output: 2, 3, 4, 5, 6
Objects have properties with values, and you'll likely want to use both in your loop. To loop over Objects, "of" can be used instead of "in". 
myObject =
    name: "Koen"
    city: "Amsterdam"
    age: 31

for key, value of myObject
    key, ":", value 

# name:Koen, city:Amsterdam, age:31
#### Conditionals Conditionals define your logic. You use them together with comparators to make decisions. They are all based around the **if**, **else if** and **else** keywords. 
	if age > 21
		console.log "Over 21!"
A conditional that checks for multiple things: 
	if age < 21
		console.log "Young"
	else if age < 65
		console.log "Adult"
	else
		console.log "Senior"
##### Comparators Comparators are used to compare things.
The most common ones are equality and greater/smaller-than. 
4 is 4 # Output: true
4 isnt 4 # Output: false
4 < 5 # Output true
4 > 5 # Output false
