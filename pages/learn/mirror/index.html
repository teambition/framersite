{% endblock sidebar %} {% block body %} {% filter pygmentify %} {% filter markdown %} ### Mirror Mirror allows you to preview your prototypes on any mobile device. It works on any device connected to the same Wi-Fi network with a WebKit-based browser, including iPhones, iPads, Android and tablets. On save, Framer Studio automatically refreshes your project for live-previewing. --- ##### Get Started To use Mirror, click the **"Mirror"** icon in the toolbar. Next, open the link presented within the dropbown on your mobile device. Use the share button to quickly send the link to yourself or others via message or email. Framer Studio runs a built-in webserver containing all of your open projects. The address is your Network **IP address**. The web server doesn't rely on Bonjour, so it works on all networks, including corporate managed networks. ---
