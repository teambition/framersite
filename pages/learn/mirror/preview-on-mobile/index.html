{% endblock sidebar %} {% block body %} {% filter pygmentify %} {% filter markdown %} ### Preview on Mobile If you've sent yourself the link to your prototype via email or text, opening it on your mobile device should automatically launch a web browser with your prototype. **On iOS**, you can add your project to your Homescreen by tapping the Share button and then **"Add to Home Screen"**. This allows you to view your prototypes without the browser chrome of Safari. To see all of your active documents, go to the root of the webserver. Simply remove your project name from the URL to get there. --- ##### Notes - The **inital-scale** meta tag is set to 0.5 by default. This works best when creating prototypes for Retina (@2x) devices. You can change this when working in @1x. - To save your prototype for offline (cached) viewing, add /offline.html to the URL. Note that his doesn't work well with auto refreshing. {% endfilter %} {% endfilter %} {% endblock body %}
