{% endblock sidebar %} {% block body %} {% filter pygmentify %} {% filter markdown %} ### Photoshop & Sketch Every layer group in Photoshop and Sketch will be imported as a layer within Framer. Hidden layers will also be imported, but will also be hidden in Framer, with opacity set to zero. --- ### Principles Framer is fully compatible with Sketch 3.1, Sketch Beta, Photoshop CS6, Photoshop CC and Photoshop CC 2014. --- ##### Photoshop & Sketch - Only saved & active documents can be imported - Every layer group will become a layer in Framer - Layer group names should be unique - Masked layer groups become layers with clip enabled
##### Sketch only - Only the first artboard will be visible in Framer by default - Items with negative coordinates will not be visible in Framer - You can only use one mask per layer group
