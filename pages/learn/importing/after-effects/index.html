Transparent Videos

With After Effects, it's possible to export fully transparant videos. These can be useful for prototypes, but note that animating transparant videos may create some jagged edges due to anti-aliasing. When possible, exporting with a solid background is recommended.

  • Toggle Transparency grid on your composition
  • Add your project to the Render Queue
  • In the Output Module, select the RGB + Alpha channel
  • In the Format Options, use the Animation Codec
--- ##### Web-compatible videos Framer and Safari support Lossless movie formats from After Effects, but the .MOV file format doesn't play well on most browsers. To create web-compatible videos, convert them to MP4 or export using Adobe Media Encoder, and use .H264 as the Codec and .MP4 as file format ---
