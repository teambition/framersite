{% endblock sidebar %} {% block body %} {% filter pygmentify %} {% filter markdown %} ### Preview Prototypes Prototypes can be previewed on any mobile device connected to the same Wi-Fi network with a WebKit-based browser. Supported browsers include Safari and Chrome on iOS and Android devices. --- ##### Get Started To start previewing your prototypes, click the **"Mirror"** icon in the toolbar. Next, open the link presented in the popover menu on your mobile device. The share button can be used to quickly send yourself or others the link via message or email.
##### How it works Framer Studio runs a built-in webserver containing all of your open projects. The address is your Network **IP address**. On save, Framer Studio automatically refreshes your project for live-previewing. The web server doesn't rely on Bonjour, so it works on all networks, including corporate managed networks. ---
