Layers are the basic containers for Framer. They have positioning, size, visual properties and a hierarchy. They can contain images, videos, text or even arbitrary html. To create a layer you use the new keyword. Layers get a set of default properties like a blue color and a width and height. You can override these in the Framer.Defaults. 
layerA = new Layer()
You can set all layer properties when you create it or after creation. 
layerA = new Layer({
	x:100, 
	height:100, 
	width:500, 
	opacity:0.5, 
	backgroundColor:"red"
})

layerA.y = 900
layerA.opacity = 1
