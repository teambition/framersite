###### layer.centerY(offset number) Center this layer vertically in it's superlayer. If there is no superlayer it will be centered relative to the screen. The offset is a pixel offset from the center and optional. 
layerA = new Layer({width:500, height:500})
layerB = new Layer({width:100, height:100, superLayer:layerA})

layerB.centerY()
print layerB.x, layerB.y
# Output: 0, 200

layerB.centerY(20)
print layerB.x, layerB.y
# Output: 0, 220
